# Project Dunj.
## Dependencies
Built under python v3.7.6 and pygame v1.9.6.
## Instructions
During each turn, players can move a set amount of tiles.
### Commands
d - drop currently equipped weapon
enter - end turn 
/ - display help menu
e - enter attack mode
| Command | Description |
| ----------- | ----------- |
| d | Text |
| enter | Text |
| / | Text |
| e | Text |
